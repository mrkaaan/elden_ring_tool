# 艾尔登法环存档管理器 (v2.0)

一个用于管理《艾尔登法环》游戏和 ModEngine 的存档管理工具。

## 功能

- **存档管理**：在 `saves` 目录中列出所有存档备份，支持一键导入或导出当前游戏存档。
- **路径检测**：自动检测游戏存档路径 (`%AppData%\Roaming\EldenRing`)，若失败可手动指定。
- **配置记忆**：“直接覆盖”等选项的设置会被保存至 `config.json`，重启后状态不变。
- **启动游戏**：点击“启动游戏”按钮，执行同目录下的 `launchmod_eldenring.bat` 文件来启动游戏。程序会检查该文件是否存在，并据此启用或禁用按钮。

## 变更

由于发现了更加好用的ModEngine，工具的核心定位已发生了变化：

- 移除了与“无缝联机MOD”相关的所有配置和管理功能（如密码、死亡惩罚设置等）。
- 启动游戏的功能从执行游戏目录下的 `ersc_launcher.exe` 改为执行同目录下的 `launchmod_eldenring.bat`。
- 现在使用 **ModEngine** 作为MOD管理器来加载和运行游戏，本工具不再处理任何MOD文件。
