1 添加一个下拉框，内容读取当前工具目录下的存档文件夹，显示名称为其下面的文件名称，只要文件存在都可以在下拉框中选中
2 存档文件夹中的格式如下
存档文件夹
- 以时间戳命名的存档文件避免冲突
    - EldenRing 文件夹
        - 存档文件夹1 这是一个以数字命名的文件夹是steamID，也就是游戏的唯一标识符 比如 76561198925622664，位数固定的，单数数字不固定
            - ER0000.sl2
            - ER0000.co2
            - ER0000.co2.bak
        - 存档文件2 GraphicsConfig.xml 文件
3 对于存档文件检验的标准
3.1 以时间戳命名的存档文件这个要做是否存在的判定 其下面可能没有EldenRing这个文件夹，可能是直接存档文件，两种情况都要做支持
3.2  SteamID的17位数字的文件夹是否存在要做判定，其下面的文件数量不做判定，只要存在ER开头的文件就行
3.3  GraphicsConfig.xml 文件不做是否存在的判定
4 所有存档文件在导入之前，先判断原本有无存档，有的话执行一次备份，放到工作根目录的存档文件夹中

6 导出存档的的时间戳文件夹名称要与覆盖存档自动备份的文件夹名称做区别，比如一个是 时间戳_存档备份，一个是 时间戳_自动存储备份，名称用英文，具体用什么你来决定
所有文件直接覆盖
7 在导入存档按钮的右侧，添加一个复选框，文字是"直接覆盖"，默认是选中的

11 如果目标存档位置有EldenRing文件夹，要做好我在3.1中的适配，避免出现把备份存档的EldenRing直接移动到目标存档位置的EledenRing文件夹，造成两个EledenRing文件夹嵌套的情况
12 在做存档备份的时候，是直接复制备份EledenRing文件夹，还是复制备份EledenRing文件夹下的所有文件，这个你来做定夺
